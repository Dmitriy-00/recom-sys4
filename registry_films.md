# Реестр значений полей - База фильмов

> **Назначение:** Стандартизированный справочник допустимых значений для полей шаблона фильмов

---

## КАТЕГОРИАЛЬНЫЕ ПОЛЯ

### asl.category (Категория среднего длины кадра)

| Значение | Описание | Диапазон (сек) | Примеры |
|----------|----------|----------------|---------|
| `hyperkinetic` | Гиперкинетический монтаж | < 2 | "Беги, Лола, беги", "Kingsman", клипы |
| `fast` | Быстрый монтаж | 2-4 | "Безумный Макс: Дорога ярости", "Социальная сеть" |
| `medium` | Средний темп | 4-6 | Большинство голливудских фильмов, "Начало" |
| `slow` | Медленный монтаж | 6-8 | "Бегущий по лезвию 2049", Таркий |
| `contemplative` | Созерцательное кино | > 8 | "Сталкер", "Древо жизни", Бела Тарр |

**Как определить:** Среднее время между монтажными склейками. Считайте склейки в репрезентативной 5-минутной сцене и делите на количество кадров.

### color_palette.dominant (Доминирующая цветовая палитра)

| Значение | Описание | Примеры |
|----------|----------|---------|
| `warm` | Преобладают тёплые тона (красный, оранжевый, жёлтый) | "Амели", "Гранд Будапешт отель", "О, где же ты, брат?" |
| `cool` | Преобладают холодные тона (синий, зелёный, фиолетовый) | "Бегущий по лезвию", "Драйв", "Луна" |
| `neutral` | Сбалансированная или приглушенная палитра | "Дорога", многие военные фильмы |
| `monochrome` | Чёрно-белая или почти монохромная | "Художник", "Франкенвини", "Белая лента" |
| `varied` | Сильная вариация по сценам/актам | "Героин", "Три цвета" (трилогия) |
| `saturated` | Высоконасыщенные цвета | "Спайдермен: Через вселенные", Уэс Андерсон |
| `desaturated` | Низконасыщенные, блеклые цвета | "Дети человечества", "Безумный Макс: Дорога ярости" |

### composition.framing_style (Стиль кадрирования)

| Значение | Описание | Признаки | Примеры |
|----------|----------|----------|---------|
| `classical` | Классическая композиция | Правило третей, симметрия, центрирование главных объектов | Голливудские студийные фильмы 40-60х |
| `modern` | Современная композиция | Баланс между классикой и экспериментом, динамичная камера | Большинство современных фильмов |
| `experimental` | Экспериментальная композиция | Нарушение правил, необычные углы, асимметрия | "Догвилль", Годар, авангард |
| `documentary` | Документальный стиль | Естественное освещение, ручная камера, минимум постановки | Догма 95, неореализм |
| `tableaux` | Театральная композиция | Статичная камера, сцена как картина | Уэс Андерсон, Рой Андерссон, "Святые моторы" |

### editing_style (Стиль монтажа)

| Значение | Описание | Примеры |
|----------|----------|---------|
| `continuity` | Непрерывный монтаж (классический) | Большинство нарративных фильмов |
| `montage` | Монтаж Эйзенштейна (контраст, столкновение) | "Броненосец Потёмкин", "Реквием по мечте" |
| `jump_cut` | Резкие склейки (французская новая волна) | "На последнем дыхании", "Социальная сеть" |
| `long_take` | Длинные планы с минимумом склеек | "Бёрдмэн", "Русский ковчег", "Гравитация" (открытие) |
| `parallel` | Параллельный монтаж | "Крёстный отец" (крещение), Нолан |
| `experimental` | Экспериментальный | "Древо жизни", "Mulholland Drive", авангард |

### pacing.act_structure (Актовая структура)

| Значение | Описание | Примеры |
|----------|----------|---------|
| `3-act` | Классическая трёхактная структура | Большинство голливудских фильмов |
| `5-act` | Пятиактная структура (шекспировская) | "Касабланка", сложные драмы |
| `2-act` | Двухактная структура | Некоторые европейские фильмы |
| `non-linear` | Нелинейная структура | "Криминальное чтиво", "Мементо", "Облачный атлас" |
| `episodic` | Эпизодическая структура | "Кофе и сигареты", "Магнолия", "Любовь" (Ханеке) |
| `circular` | Циклическая структура (конец=начало) | "Двойная страховка", "Донни Дарко" |
| `fragmented` | Фрагментированная структура | "Дерево жизни", "Синекдоха, Нью-Йорк" |

### emotional_arc.type (Тип эмоциональной траектории)

**Основано на исследовании Reagan et al., 2016**

| Значение | Описание | График | Примеры |
|----------|----------|--------|---------|
| `rags_to_riches` | От грязи к богатству | ↗️ Рост | "Rocky", "Форрест Гамп", "Король Лев" |
| `riches_to_rags` | Трагедия, падение | ↘️ Падение | "Гражданин Кейн", "Нефть", "Волк с Уолл-стрит" |
| `man_in_hole` | Человек в яме | ↘️↗️ Падение→Рост | "127 часов", "Побег из Шоушенка", "В поисках Немо" |
| `icarus` | Икар | ↗️↘️ Рост→Падение | "Птица", "Чёрный лебедь", "Во все тяжкие" (сериал) |
| `cinderella` | Золушка | ↗️↘️↗️ Рост→Падение→Рост | "Гарри Поттер", "Золушка", большинство приключений |
| `oedipus` | Эдип | ↘️↗️↘️ Падение→Рост→Падение | "Лунный свет", "Фарго" (сериал) |
| `flat` | Ровная эмоциональная траектория | → Без динамики | Некоторые созерцательные фильмы |
| `complex` | Сложная траектория | 〰️ Много пиков | "Облачный атлас", эпические саги |

### character_archetypes (Архетипы персонажей)

**Основано на Юнге, Кэмпбелле, Воглере**

| Архетип | Описание | Функция в истории | Примеры |
|---------|----------|-------------------|---------|
| `Hero` | Герой | Протагонист, проходящий путь трансформации | Люк Скайуокер, Нео, Фродо |
| `Mentor` | Наставник | Даёт знания и инструменты герою | Оби-Ван, Морфеус, Гэндальф |
| `Threshold_Guardian` | Страж порога | Проверяет готовность героя | Швейцар в "Волшебнике из страны Оз" |
| `Herald` | Вестник | Приносит призыв к приключению | R2-D2 с посланием Леи |
| `Shapeshifter` | Перевёртыш | Вносит неопределённость, меняет облик | Женские персонажи в нуаре, Снейп |
| `Shadow` | Тень | Антагонист, тёмная сторона героя | Дарт Вейдер, Джокер, Голлум |
| `Trickster` | Трикстер | Нарушает правила, комическое облегчение | Хан Соло, Джек Воробей, Локи |
| `Ally` | Союзник | Поддерживает героя | Сэм Гэмджи, Гермиона |

**Примечание:** Один персонаж может совмещать несколько архетипов или менять их по ходу истории.

### moral_ambiguity (Уровень моральной неопределённости)

| Диапазон | Описание | Примеры |
|----------|----------|---------|
| `0.0-0.2` | Чёрно-белая мораль | Дисней, ранний Голливуд, детские фильмы |
| `0.3-0.4` | Преимущественно ясная мораль с нюансами | Большинство супергеройских фильмов, Marvel |
| `0.5-0.6` | Умеренная амбивалентность | "Крёстный отец", "Гладиатор" |
| `0.7-0.8` | Высокая амбивалентность | "Нефть", "Паразиты", "Джокер" |
| `0.9-1.0` | Полная моральная неопределённость | "Сербский фильм", "Смешной игры", провокативное кино |

### perspective_types (Типы перспективы повествования)

| Значение | Описание | Примеры |
|----------|----------|---------|
| `single` | Единая перспектива | Большинство фильмов с одним протагонистом |
| `multiple` | Множественные перспективы | "Магнолия", "Вавилон", "Облачный атлас" |
| `switching` | Переключающиеся перспективы | "Бонни и Клайд", "Gone Girl" |
| `omniscient` | Всезнающий рассказчик | Фильмы с закадровым повествованием |
| `unreliable` | Ненадёжный рассказчик | "Бойцовский клуб", "Обычные подозреваемые", "Мементо" |
| `subjective` | Субъективная камера (POV) | "Хардкор", "Леди в озере" |

### target_cognitive_style (Целевой когнитивный стиль)

| Стиль | Описание | Предпочтения в кино | Примеры фильмов |
|-------|----------|---------------------|-----------------|
| `visual` | Визуальный тип | Медленный монтаж (ASL>5), высокая визуальная сложность, символизм через образы | "Бегущий по лезвию 2049", Тарковский, Малик |
| `analytical` | Аналитический тип | Сложный нарратив, множественные POV, интеллектуальные головоломки | "Начало", "Primer", Нолан, Кауфман |
| `empathetic` | Эмпатический тип | Глубокие персонажи, эмоциональные траектории, фокус на отношениях | "Лунный свет", "Брак" (Marriage Story), драмы |
| `kinesthetic` | Кинестетический тип | Быстрый монтаж (ASL<4), динамичное действие, физическая вовлечённость | "Безумный Макс", экшн, триллеры |

**Примечание:** Фильм может подходить нескольким стилям одновременно.

### difficulty_level (Уровень сложности)

| Уровень | Описание | Требует | Примеры |
|---------|----------|---------|---------|
| `1-2` | Очень лёгкий | Минимум внимания, понятен всем | Детские фильмы, простые комедии |
| `3-4` | Лёгкий | Базовое следование за сюжетом | Большинство блокбастеров |
| `5-6` | Средний | Внимание к деталям, базовая интерпретация | "Начало", "Остров проклятых" |
| `7-8` | Сложный | Активное мышление, знание контекста | "Мулхолланд Драйв", Кауфман, артхаус |
| `9-10` | Очень сложный | Подготовка, множественные просмотры, знание теории | "Синекдоха Нью-Йорк", экспериментальное кино |

---

### best_viewing_context (Оптимальный контекст просмотра)

| Контекст | Описание | Почему |
|----------|----------|--------|
| `solo_focused` | Индивидуальный, концентрированный | Требует полного внимания, личная рефлексия |
| `solo_relaxed` | Индивидуальный, расслабленный | Можно смотреть фоном или без напряжения |
| `group_discussion` | Группа с обсуждением | Выигрывает от множественных интерпретаций |
| `group_experience` | Группа для совместного опыта | Эмоции усиливаются в группе |
| `academic` | Академический контекст | Нужен анализ, комментарии, подготовка |
| `cinematic` | Кинотеатр | Технически требователен (звук, изображение) |

## КОГНИТИВНЫЕ ОПЕРАЦИИ (cognitive_operations)

### 8 атомарных операций метасистемы

```yaml
operations:
  - name: Pause
    description: Остановка автоматических реакций
    activators:
      - Медленный монтаж
      - Длинные планы
      - Созерцательные сцены
      - Паузы в диалогах
    
  - name: Observation
    description: Внимательное наблюдение без суждений
    activators:
      - Документальный стиль
      - Субъективная камера
      - Визуальные детали
      - Показывание без объяснения
    
  - name: Decomposition
    description: Разбиение на компоненты
    activators:
      - Нелинейный нарратив
      - Параллельный монтаж
      - Множественные перспективы
      - Сложная структура
    
  - name: Linkage
    description: Поиск связей и паттернов
    activators:
      - Параллели между сюжетными линиями
      - Символические повторы
      - Тематические мотивы
      - Причинно-следственные цепи
    
  - name: Separation
    description: Различение связанного и несвязанного
    activators:
      - Развенчание иллюзий
      - Разоблачение манипуляций
      - Критика ложных связей
      - Деконструкция стереотипов
    
  - name: Testing
    description: Проверка предположений
    activators:
      - Моральные дилеммы
      - Контрфактуальные сценарии
      - Испытание убеждений
      - Эксперименты персонажей
    
  - name: Integration
    description: Синтез нового понимания
    activators:
      - Разрешение парадоксов
      - Кульминационные озарения
      - Трансформация персонажей
      - Финальный синтез
    
  - name: Reflection
    description: Метакогнитивное осмысление
    activators:
      - Метафильмические элементы
      - Разрушение четвёртой стены
      - Рефлексивные монологи
      - Осознание медиума
```

## СТАТУСЫ ВАЛИДАЦИИ (validation_status)

| Статус | Описание | Требования |
|--------|----------|------------|
| `draft` | Черновик | Базовые поля заполнены, требует проверки |
| `peer_reviewed` | Проверено коллегами | 2+ аннотатора согласны |

| `expert_reviewed` | Проверено экспертом | Подтверждено кинокритиком/исследователем |

## РЕКОМЕНДАЦИИ ПО ИСПОЛЬЗОВАНИЮ

### Заполнение полей

1. **Обязательные поля:**
   - title, year, director, country
   - asl.category, difficulty_level
   - genres (минимум 1)
   - transformative_score

2. **Рекомендуемые поля:**
   - emotional_arc.type
   - target_cognitive_style
   - themes (3-5)
   - cognitive_operations_activated

3. **Опциональные поля:**
   - Все остальные заполняются по возможности

### Обеспечение качества

- **Inter-rater reliability:** Минимум 2 аннотатора для субъективных полей
- **Калибровка:** Периодическое обсуждение спорных случаев
- **Эталоны:** 20-30 эталонных фильмов с экспертной разметкой
- **Обучение:** Новые аннотаторы начинают с простых фильмов
